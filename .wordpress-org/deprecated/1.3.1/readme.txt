自定义微信分享链接中的小图标。

== Description ==
此插件可以自定义分享到朋友和微信朋友圈的链接中的小图标。未来版本将通过微信API实现更多信息的自定义。

== Installation ==
下载插件
上传到目录 /wp-contents/plugins/
在WordPress“插件”中激活本插件

= Configuration =
设置 -> 微信分享，选中你要自定义小图标的文章类型并保存设置。文章，页面和媒体是默认选中的文章类型。

== Frequently Asked Questions ==
为什么在其它文章类型中找不到设置小图标的地方？
